# 프로젝트 정보
1. 설명
   * localStorage를 이용해 진행 단계를 저장 가능하도록 구현 했습니다.
   * 총 9마리의 공룡에 대한 정보로 1~9까지의 단게로 구성 되어 있습니다.
   * quiz를 풀고 마지막에 나오는 공룡들을 꼭 봐주세요~
      * 귀찮으신 분들은 [공룡 바로 보기](https://handeul-kim.github.io/project-kiosk/wg01.html) 01~09.html
   
2. 참고
